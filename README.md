Este repositório é mais do que um currículo online: ele funciona como uma vitrine técnica para recrutadores, líderes de tecnologia e empresas que buscam alguém capaz de entender o problema, desenhar a solução, desenvolver, publicar e evoluir o produto.

> Perfil full cycle com visão de negócio, autonomia técnica e foco em entregar soluções que reduzem gargalos operacionais.

## Como rodar localmente

```bash
npm install
npm run dev
```

Para gerar e visualizar a versão de produção:

```bash
npm run build
npm run preview
```

O build final é gerado em `dist/`.

## Deploy

O projeto está preparado para publicação via **GitHub Pages**. O workflow executa:

```bash
npm ci
npm run build
```

Depois, publica automaticamente a pasta `dist` quando há push na branch `main`.
